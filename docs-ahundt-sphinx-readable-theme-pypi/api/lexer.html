xonsh.lexer.handle_token(state, token)[source]¶

General-purpose token handler. Makes use of token_map or special_map to yield one or more PLY tokens from the given input.

Parameters:
  • state – The current state of the lexer, including information about whether we are in Python mode or subprocess mode, which changes the lexer’s behavior. Also includes the stream of tokens yet to be considered.
  • token – The token (from tokenize) currently under consideration
